# Emotion Recognition from Speech

## Project Overview

This project detects human emotions from speech audio using Deep Learning and MFCC audio features.

## Technologies Used

* Python
* TensorFlow
* Keras
* Librosa
* NumPy
* Scikit-learn

## Dataset

RAVDESS Emotional Speech Audio Dataset

## Features

* Audio feature extraction using MFCC
* Emotion classification
* Neural Network model
* Model saving using Keras

## Model Accuracy

Achieved approximately 69% test accuracy.

## How to Run

```bash
python main.py
```

## Output

The trained model is saved as:

```bash
emotion_model.keras
```
